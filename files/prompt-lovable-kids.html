Prompt principal
## OVELHINHA — SISTEMA DE PULSEIRAS KIDS CHURCH 🐑 Crie um sistema web completo chamado "Ovelhinha" para igrejas gerenciarem a área kids com pulseiras de LED. ### CONTEXTO Pais deixam seus filhos na área kids da igreja e recebem uma pulseira numerada. Quando há algum problema, a equipe aciona a pulseira do pai pelo sistema, que acende o LED da pulseira dele. O pai vai até a recepção. O slogan do sistema é: "Cada criança, no lugar certo." ### STACK TÉCNICA - React + Vite + TypeScript - Tailwind CSS para estilização - React Router para navegação entre telas - Zustand para estado global - React Hook Form + Zod para formulários - Recharts para gráficos no dashboard - Dados em localStorage por enquanto (simulando banco de dados) ### IDENTIDADE VISUAL — OVELHINHA O sistema tem identidade visual própria. Siga rigorosamente: Nome: Ovelhinha Tagline: "Cada criança, no lugar certo." Tom: Fofo mas profissional — acolhedor para igrejas, confiável para pais Paleta de cores (use exatamente esses valores): - Primária · Azul Céu: #5B8CFF → botões principais, header, links, destaques - Secundária · Âmbar: #FFB347 → pulseira, badges, elementos de destaque caloroso - Sucesso · Verde Mint: #3ECFAA → confirmações, "pai chegou", status OK - Urgência · Coral: #FF6B6B → chamadas abertas, alertas, urgência - Background · Lã: #F0EDE8 → fundo das telas, cards neutros - Background alt: #F8F9FC → fundo principal das páginas - Texto principal: #1A1F36 - Texto secundário: #6B7280 - Borda/divisor: #E8EAF0 - Branco: #FFFFFF → cards, superfícies Tipografia: - Títulos e logo: fonte Nunito (Google Fonts) — peso 800 e 900, letter-spacing -0.02em - Corpo e interface: fonte DM Sans (Google Fonts) — peso 400 e 500 - Números e códigos: fonte DM Mono (Google Fonts) — ex: #07, ESP32:A4:B2 Logo: - Exibir o emoji 🐑 ao lado do nome "Ovelhinha" em Nunito 900 - Tagline abaixo em DM Sans 500, tamanho menor, cor #6B7280 - No header do sistema: fundo #5B8CFF, logo branco Estilo dos componentes: - Border-radius generoso: cards 16px, botões 10px, inputs 10px - Sombras suaves: box-shadow 0 2px 12px rgba(0,0,0,0.06) - Botão primário: fundo #5B8CFF, texto branco, hover #3D6FE8 - Botão de urgência: fundo #FF6B6B, texto branco, animação pulse quando chamada ativa - Ícones: pacote lucide-react - Inputs: borda #E8EAF0, focus borda #5B8CFF com glow suave azul ### TELAS DO SISTEMA 1. TELA DE LOGIN - Logo Ovelhinha centralizado (emoji 🐑 + nome em Nunito 900 + tagline) - Fundo com cor #F0EDE8 (lã) com padrão sutil de bolinhas ou textura suave - Dois cards clicáveis para selecionar perfil: "Recepção" (ícone 🖥️) e "Tia da Sala" (ícone 📱) - Campo de senha para Recepção; campo de código de 4 dígitos para Tia - Botão "Entrar" em #5B8CFF 2. DASHBOARD — RECEPÇÃO (rota: /dashboard) - Header azul (#5B8CFF) com logo Ovelhinha branco à esquerda e nome da igreja à direita - Sidebar esquerda branca com navegação: 🏠 Dashboard, ➕ Novo Cadastro, ⌚ Pulseiras, 📊 Relatórios, ⚙️ Configurações - 4 cards de resumo no topo com ícones: crianças presentes (azul), pulseiras ativas (âmbar), chamadas abertas (coral piscando), bateria baixa (amarelo) - Seção "Chamadas Abertas" com cards em fundo #FFF5F5, borda coral, animação pulse no indicador vermelho - Lista de crianças presentes com status colorido (presente=verde, chamado=coral, saiu=cinza) 3. NOVO CADASTRO (rota: /cadastro) - Stepper visual com 3 passos em #5B8CFF para o passo ativo - Passo 1 — Dados da criança: nome, data de nascimento, sala, observações médicas - Passo 2 — Responsável: nome, telefone, botão "+ Adicionar segundo responsável" - Passo 3 — Pulseira: campo grande para digitar número (ex: 07), badge âmbar "#07" aparece ao digitar, confirmação "Pulseira #07 vinculada a João Silva ✓" em verde - Tela de sucesso: confete animado, opções "🖨️ Imprimir Etiqueta" e "📋 Só o número" - Etiqueta de impressão: nome da criança em Nunito 900 grande, número #44 em DM Mono, QR Code simulado, sala e data 4. ACIONAR PAI (rota: /acionar) - Campo de busca grande no topo com ícone de lupa e placeholder "Nome ou número da criança..." - Botão de câmera (📷) ao lado do campo para simular QR Code - Resultados em tempo real como cards com avatar circular colorido por sala - Card da criança selecionada: nome grande, número em badge âmbar, sala, nome do pai, número da pulseira - Seletor de motivo em grid 2x3 com ícones grandes e fundo colorido ao selecionar: 🚽 Banheiro · 🤒 Passando mal · 😢 Chorando · ⚠️ Urgência · 🍼 Amamentação · 📝 Outro - Botão "CHAMAR PAI" coral (#FF6B6B), largura total, altura 56px, Nunito 800 - Estado "Aguardando...": cronômetro em destaque, barra de progresso coral, botões "✓ Pai Chegou" (verde) e "🔁 Reacionar" (âmbar) 5. GESTÃO DE PULSEIRAS (rota: /pulseiras) - Grid de cards 4 colunas, um por pulseira - Cada card: número em DM Mono grande (#01), badge de status colorido, barra de bateria, nome do pai (se em uso) - Status: 🟢 Disponível (#3ECFAA) · 🟠 Em uso (#FFB347) · 🔵 Carregando (#5B8CFF) · ⚫ Offline (#6B7280) - Filtros em pills no topo: Todas · Disponíveis · Em uso · Bateria baixa - Botão "+ Registrar pulseira" em #5B8CFF 6. APP MOBILE — TIA DA SALA (rota: /tia) - Interface max-width 430px, otimizada para celular - Header compacto azul com "🐑 Ovelhinha" e nome da sala - Botão enorme de câmera centralizado: círculo azul grande "📷 Escanear etiqueta" - Campo de busca abaixo - Lista de crianças da sala com avatar, nome e botão "Chamar Pai" coral em cada linha - Confirmação: tela cheia verde mint por 3s com "✅ Pai de João foi chamado!" em Nunito 900 - Notificação de chegada: banner verde deslizando do topo "Pai de João chegou · 10:32" 7. RELATÓRIOS (rota: /relatorios) - Filtro de período no topo - Cards de métricas com ícones e números em Nunito 900 azul - Gráfico de barras (Recharts) com cor #5B8CFF: crianças por culto - Gráfico de pizza: motivos das chamadas com as cores da paleta - Tabela zebrada com histórico de chamadas, botão exportar CSV 8. CONFIGURAÇÕES (rota: /configuracoes) - Seções separadas por cards: Igreja, Salas, Usuários, Sistema - Toggle switches estilizados em #5B8CFF - Campo para nome da igreja e upload de logo - Lista de salas com botão editar/excluir - Gerador de código do dia: botão gera código de 4 dígitos e exibe em destaque para a tia copiar - Slider para tempo de reacionamento (1 a 15 minutos) ### DADOS SIMULADOS Popule com dados brasileiros realistas: - Igreja: "Igreja Graça e Vida" - 3 salas: "🍼 Berçário (0-2 anos)", "🌟 Maternal (3-5 anos)", "✨ Infantil (6-10 anos)" - 12 crianças com nomes brasileiros (João, Maria, Pedro, Ana, Lucas, Beatriz...) - 3 pulseiras em uso (#03, #07, #12), 17 disponíveis - 2 chamadas abertas: uma "Chorando" há 3 min, uma "Banheiro" há 1 min - Histórico de 4 cultos para relatórios ### COMPORTAMENTOS IMPORTANTES - Ao clicar "CHAMAR PAI": animação de loading "Acionando pulseira #07..." por 1.5s, depois confirma com som visual - Cronômetro nas chamadas abertas em tempo real - Após 5 minutos sem resposta: card pisca em coral, alerta no dashboard - Busca filtra em tempo real sem Enter - Formulário valida cada etapa antes de avançar - Toast notifications com ícone 🐑: "Pulseira #07 acionada! 🐑", "Cadastro salvo! ✓" - Transições suaves entre telas (fade 200ms) - Loading skeletons nos cards enquanto "carrega" dados
